
My research focuses on **Trustworthy Code Generation** for industrial-level complex programming tasks. I am particularly interested in leveraging deep learning and reinforcement learning techniques to improve the reliability and interpretability of code generation systems.

#### Research Projects

**Trustworthy Code Generation for Industrial-Level Complex Programming Tasks**
*National Innovation Training Project (Principal Investigator), 2025-present*

This project addresses three core challenges in trustworthy code generation:
- **Multimodal Semantic Representation**: Developing comprehensive representations that capture both syntactic and semantic properties of code
- **Multi-stage Planning**: Enhancing algorithmic skill acquisition through hierarchical planning mechanisms
- **Interpretable Decision Chains**: Making the code generation process traceable and explainable

The research aims to advance the state-of-the-art in automated programming assistance, with applications in industrial software development and educational tools.

#### Research Interests

- **Deep Learning** for Code Representation
- **Reinforcement Learning** for Program Synthesis
- **Multi-Agent Systems** for Collaborative Coding
- **Trustworthy AI** in Software Engineering
